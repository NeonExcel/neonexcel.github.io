FAQ

How can I join the servers?
Go to the servers tab, scroll down and select the server you want to join, expand the box, click on the hyperlink, and look for instuctions on that page.
Is there a way to contact support?
Yes. You can join the discord server and submit your issue in the support channel.
Is there a way I contribute?
If you would like to contribute, you can contact me on discord.
Where can I find the server schedules?
You can find when the servers are online on the page for that server.
Is there a way I can report issues?
You can report issues by going to the report or issues channel on the discord server.
